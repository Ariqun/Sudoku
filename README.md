# :video_game: Игра Судоку #

### :pencil: Описание ###

Классическое судоку.

В любой непонятной ситуации прогресс текущей игры сохранется в localStorage.

Есть возможность внести свое судоку, нажать на кнопочку, и грёмлины, крутящие педали в процессоре, быстренько решат его и предоставят вам. Без смс и регистрации.

##### :arrow_forward: Запуск #####

`npm start`

##### :fast_forward: Production версия #####

`npm run build`
